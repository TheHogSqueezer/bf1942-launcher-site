Install Guide / FAQ

Everything you need to know

This page covers the most common questions about setup, launcher updates, and getting into Battlefield 1942 multiplayer quickly.

Which installer should I choose?

Use Community Package Setup if you want the broader community setup path. Use Launcher Setup if you already have Battlefield 1942 configured and only need the launcher.

What does the Launcher Setup do?

The Launcher Setup is intended for players who only need the BF1942 launcher and server browser, especially for existing installs or launcher updates.

What is the Community Package Setup?

The Community Package Setup is the broader community-oriented setup path intended to help players get into multiplayer with a more complete installation flow.

How do I join a server?

Open the BF1942 Launcher, let the server list refresh, then double click a server to join.

How do launcher updates work?

If you already have Battlefield 1942 configured, the Launcher Setup is the intended path for getting the newest launcher release.

What should I do after installing?

  • Open the BF1942 Launcher from the desktop or Start Menu.
  • Confirm your path or settings if needed.
  • Wait for the server browser to refresh.
  • Double click a server to play.

Is this an official EA or DICE project?

No. PlayBF1942 is a community project and is not affiliated with or endorsed by Electronic Arts or DICE.
