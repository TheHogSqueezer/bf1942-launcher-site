Install Guide / FAQ

Everything you need to know

This page covers the most common questions about installation, launcher updates, and getting into Battlefield 1942 online quickly.

Which installer should I choose?

Use Community Package Setup if you want the full Battlefield 1942 install. Use Launcher Setup if you already have BF1942 installed and only need the launcher.

Does the Community Package include the expansions?

Yes. The Community Package includes Battlefield 1942, Road to Rome, and Secret Weapons.

Do I need to manually apply a multiplayer patch?

No. The goal of this package is to include the multiplayer connectivity fix so the game is already prepared for community online play.

I already have Battlefield 1942 installed. What should I download?

Download the Launcher Setup. That is intended for players who already have the game installed and only need the launcher or a launcher update.

How do I join a server?

Open the BF1942 Launcher, wait for the server list to load, then double click a server to join.

Is this an official EA release?

No. This is a community project intended to make Battlefield 1942 easier to install and play online.

What should I do after installing?

  • Open the BF1942 Launcher from the desktop or Start Menu.
  • Confirm your game path if needed.
  • Let the server browser refresh.
  • Double click a server to play.
