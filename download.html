Download Launcher Setup FAQ

Which installer should I choose?

  • Choose Community Package Setup if you do not already have Battlefield 1942 installed.
  • Choose Launcher Setup if you already have BF1942 installed and only need the launcher.
  • Choose Launcher Setup if you're updating to a newer launcher release.

What happens after install?

  • Launch the BF1942 Launcher from the desktop or Start Menu.
  • Wait for the server list to load.
  • Double click a server to join.
  • Use the launcher for future updates and server browsing.
